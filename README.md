# Simple Snake Game

This is a simple Snake game for PC using Python and Pygame.

## How to Run

1. Make sure you have Python 3.x installed.
2. Install the required dependencies:
   ```bash
   pip install pygame
   ```
3. Run the game:
   ```bash
   python snake.py
   ```

## Controls
- Arrow keys: Move the snake
- Esc: Quit the game

Enjoy!
